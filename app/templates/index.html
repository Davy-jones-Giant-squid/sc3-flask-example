Hopper: Develop web applications now

Rear Admiral Grace Hopper

Develop Python and PHP web applications with this zippy, low-memory virtual appliance. Hopper features:

  • Modern GUI: Dock, shadows and smooth effects.
  • Awesome toolchain: A baseline of useful tools like LAMP, Python, and Postgres.
  • Mac keybindings that don't stink
  • Low memory footprint: Hopper happily runs Vim, Firefox + Firebug, and a terminal in under 512mb memory.
Download Hopper virtual appliance

I'm tired of all the time that is spent by curious, fledging developers wrestling with their systems. Fighting with apt or brew often stands in the way of writing "hello world" or contributing to an open source project. I've long wanted a turnkey development environment, and I think with Hopper I'm finally close.

Install

  1. Download and install VirtualBox.
  2. Download Hopper virtual appliance
  3. Install the Hopper virtual appliance: Double click Hopper download icon, or use VirtualBox's import feature. Fill out virtual machine details, or continue to use defaults.
